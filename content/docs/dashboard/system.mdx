The System page reports runtime details: update status (current / available), database and proxy state, the log-write queue, and GPU poller / device status. It also exposes manual maintenance actions — request-log pruning and SQLite compaction.

See [Updates & health](/docs/operations/updates) and [Log retention](/docs/operations/retention).
