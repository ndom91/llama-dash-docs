---
title: Requests
description: Searchable request history with filters, histogram, and detail.
---

The Requests page is a searchable, sortable log of every proxied exchange, with a req/s histogram and filters for routing, attribution, and client host. Each row links to a detail view showing the matched routing rule, attribution metadata, latency, token counts, and estimated cost.

![Request detail](/screenshots/request-details.png)

See [Request logging](/docs/concepts/request-logging) for what is captured and how long it is kept.
