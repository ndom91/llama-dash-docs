---
title: Getting started
description: Run llama-dash in front of your inference backends.
---

llama-dash runs on port `3000` and proxies any OpenAI- or Anthropic-compatible client to llama-swap or a direct `/v1` upstream.

Full installation steps are documented in the [GitHub repository](https://github.com/ndom91/llama-dash). Point your client's base URL at the gateway and use a llama-dash API key:

```bash
export OPENAI_BASE_URL=http://localhost:3000/v1
export OPENAI_API_KEY=ld-...
```
